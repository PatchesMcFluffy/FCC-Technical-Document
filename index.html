Introduction

Lua was created as a light weight, high-level programming language to meet the increasing demand for software customization. Lua is also cross-platform due to the fact that it is interpreter is written in ANSI C. This allows developers to embed Lua via simple C API.

Lua does a few things differently than most high-level languages.

  • Lua does not include complex features by default.
  • Lua includes a set of mechanics to allow developers to add custom features to the language

Hello World!

The first thing anyone learning a programming language writes is the classic "Hello World!" program

Keeping with this tradition, this is how the "Hello World!" program is written in lua.

print("Hello World!")
Variables

Variables is lua follow similar rules to other programming languages.

  • Variables are dynamically typed, there for typecasting is not required
  • Variables do not need to be predefined before they can be used, and are initialized with a value of nil
  • Variables cannot begin with a digit
  • Variables can be any combination of letters, underscores, and digits
  • Variable are case sensitive ie: Foo, and foo are seperate and distinct variables
  • Variables are global by default
  • Local variables have lexical scope

Declaring Variables

Lua allows the declaration of local and global variables. Local variables are declared with the local prefix. 

          local foo = 2

Locally declared variables are scoped to the block they are declared in. 

          while foo<=bar do
            local bar = foo*2  --> scoped to the while loop
            print(bar)
            foo = foo + 1
            end

Global variables, like local variables, do not need to be declared in advance. You just need to assign them a value to create them. 

          print(foo) --> value of foo is nil
          foo = 10
          print(foo) --> value of foo is 10

Variables can also be deleted by assigning them the value of nil

For loops

As with other programming languages, lua has several different types of loops. We will look at a few of the most common types of loops, and what their syntax looks like.

The for loop

The for loop has two variants: the numeric for and the generic for. the numeric for loop has the following syntax: 

        for var=exp1,exp2,exp3 do
          something
        end

The loop executes something for each value of var starting with exp1, then exp2, using exp3 as the step increment. The inclusion of a step value in the loop is optional. if a value is not present, lua will assume a value of 1 for the step value. A typical numeric for looks as follows: 

        for i=1,f(x) do print(i) end

        for i=10,1,-1 do print(i) end

There are a few things that you should keep in mind when writting for loops:

  • Lua evaluates all of the values once before the loop starts
  • The control value i is a local variable decalred by the for loop, and only visible in side the loop

Generic for loop

In the example below we are printing all values of array a 

        for i,v in ipairs(a) do print(v) end

For each step is the above example, i gets an index while v gets the value associated with that index. A similar example of how to print all the keys of a table t would look like this: 

        for k in pairs(t) do print(k) end
