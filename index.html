Introduction

Lua was created as a light weight, high-level programming language to meet the increasing demand for software customization. Lua is also cross-platform due to the fact that it is interpreter is written in ANSI C. This allows developers to embed Lua via simple C API. Lua does a few things differently than most high-level languages.

  • Lua does not include complex features by default.
  • Lua includes a set of mechanics to allow developers to add custom features to the language

Hello World!

The classic "Hello World!" program is written as follows in Lua.

print("Hello World!")
Variables

Variables is lua follow similar rules to other programming languages.

  • Variables are dynamically typed, there for typecasting is not required
  • Variables do not need to be predefined before they can be used, and are initialized with a value of nil
  • Variables cannot begin with a digit
  • Variables can be any combination of letters, underscores, and digits
  • Variable are case sensitive ie: Foo, and foo are seperate and distinct variables
  • Variables are global by default
  • Local variables have lexical scope

Declaring Variables

Lua allows the declaration of local and global variables. Local variables are declared with the local prefix. 

          local foo = 2

Locally declared variables are scoped to the block they are declared in. 

          while foo<=bar do
            local bar = foo*2  --> scoped to the while loop
            print(bar)
            foo = foo + 1
            end

Global variables, like local variables, do not need to be declared in advance. You just need to assign them a value to create them. 

          print(foo) --> value of foo is nil
          foo = 10
          print(foo) --> value of foo is 10

Variables can also be deleted by assigning them the value of nil
